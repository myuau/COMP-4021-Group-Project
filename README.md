# COMP-4021-Group-Project
COMP 4021 Group Project -- Crazy McDonald

# Installation
<br>
1. install dependencies
<br>
For <b>Client</b> side:
<code>
cd client
npm install
</code>
<br>
For <b>Server</b> side:
<code>
cd server
npm install
</code>
<br>
2. Run the project
For <b>Client</b> side:
<code>
node index.js
</code>
For <b>Server</b> side:
<code>
node server.js
</code>
<br>
You can visit the client side with "localhost:3000", and visit the server side with "localhost:8000"
